Installation API

Installation

npm install prism-inventory-system
import { Inventory, Slot } from 'prism-inventory-system';

const inv = new Inventory({ size: 32, stackLimit: 99 });
inv.on('add', e => console.log('Item ajouté', e));

Galerie

Concepts clés

  • Inventory : conteneur principal avec events.
  • Slot : position pouvant contenir zéro ou un stack.
  • Stack : item + quantité (merge auto selon règles).
  • Rules : validation type, catégorie, poids.
  • Persistance : sérialisation JSON rapide.

API rapide

MéthodeDescription
add(item, qty)Ajoute un item (fusionne si possible).
remove(itemId, qty)Retire quantité spécifiée.
find(predicate)Recherche d'items.
serialize()Retourne état JSON compact.
load(json)Recharge un état sérialisé.
on(event, cb)Écoute (add, remove, move, change).
